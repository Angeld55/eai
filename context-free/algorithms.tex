\section{Алгоритми}

Тук ще докажем, че съществуват {\em полиномиални} алгоритми, които за дадена безконтекстна граматика $G$ проверяват:
\begin{itemize}
\item
  дали $\L(G) = \emptyset$;
\item
  дали $|\L(G)| = \infty$;
\item
  дали $|\L(G)| < \infty$;
\item
  по дадена дума $\alpha$ дали $\alpha \in \L(G)$.
\end{itemize}
